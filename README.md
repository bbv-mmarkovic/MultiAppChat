# MultiAppChat
Diffrente Frontend Clients Technologies interact in a Chat Application.

# Installation

## .NET Core Backend API

`dotnet restore`

## Angular Frontend

`npm install`

# Execution

## .NET Core Backend API

`dotnet run`

## Angular Frontend

`ng serve`
